What Is AttendX

Everything you need,
nothing you don't.

AttendX is a smart, real-time attendance system built for teachers. No paper, no roll-calls — just a code and 60 seconds.

📚
Subject-Based
Each subject you teach has its own card. Select it to manage attendance for that specific class — theory or lab.
🔑
One-Time Code
A unique 6-character code is generated for every session. Students enter this code to mark themselves present.
⏱️
60-Second Window
Once you start, students have exactly 60 seconds to submit the code. After that, the code expires and no one else can submit.
📋
Records Anytime
Browse attendance records by date for any subject. View who was present and at what time — all saved automatically.

Step Five — Optional

Want to review who attended on a specific day? Tap Check Attendance inside any subject panel. A full-screen sheet will open with a date picker. Choose any date and tap Fetch Records. Pro Tips

Make attendance
even smoother

Small habits that save time and avoid confusion in the classroom.

📣

Announce the code clearly

Before tapping Start, read out the code or show your screen. Students need to be ready — 60 seconds goes fast.

🔄

Regenerate if needed

If you accidentally revealed the code early or want to start fresh, tap "New Code" before hitting Start.

⚡

One session at a time

Always complete one session before starting another subject. Each Start saves data independently.

📅

Check records regularly

Use "Check Attendance" at the end of the week to review patterns. You can check any past date anytime.

🔴

Lab vs Theory — different cards

Each subject type has its own card with a colored dot — pink for Lab, green for Theory. Always select the correct one.

🛡️

The timer protects you

Since the code expires in 60 seconds, students cannot mark attendance after class ends. It's tamper-resistant by design.

FAQ

Common questions
answered

Everything teachers typically ask when starting with AttendX.

What happens if I tap Start Attendance by mistake?
The session will begin and the code will be saved. Students will see it for 60 seconds. If no one submits, no records are created for that session. You can start a new session right after by tapping Start Attendance again to generate a fresh code.
Can I extend the 60-second window?
No — the 60-second limit is fixed and intentional. It prevents students from submitting after the class period. If more time is needed, simply generate a new code and start a fresh session immediately.
How do students see the code?
Once you tap Start Attendance, the code is saved to the database and automatically appears in the students' AttendX app. Students just need to have the Student Panel open — they'll see the code and a 60-second countdown to submit.
What does "New Code" do exactly?
Tapping "New Code" generates a fresh random 6-character code and displays it — but does NOT start a session or save anything yet. Use it if you want to change the code before starting. Only tapping "Start Attendance" triggers the 60-second countdown and saves to the database.
Can I take attendance for multiple subjects in one day?
Yes! Each subject card works independently. You can take DS Lab attendance at 9 AM, then OS Theory at 11 AM — they're stored separately. Check Attendance for each subject shows its own records.
Where is the attendance data stored?
All attendance records are saved in the cloud database in real time. This means even if you close the app or switch devices, your records are safe and retrievable by date using the "Check Attendance" feature.
